# JAVA STUDY
<br>
<h2>🗒️ 프로젝트 개요</h2>

사운드 클라우드를 벤치마킹한 JSP 프로젝트.
다양한 사람들이 자신의 음악을 공유하고 대중들과 소통할 수 있는 기회의 장을 만들고, 이용자들은 자신의 취향에 맞는 다양한 음악을 들을 수 있도록 만들었습니다.

## 🚩 프로젝트 기간 & 팀원

- 2024년 3월 5일 - 2023년 3월 15일 (10일)
- 최정현(팀원)
